# Plant Disease Detection Project

This directory contains the source code for the plant disease detection system.

## Files Overview

- `model.py`: Main CNN model implementation with PlantDiseaseDetector class
- `data_preprocessing.py`: Data loading, preprocessing, and analysis utilities
- `evaluation.py`: Model evaluation, visualization, and performance metrics
- `utils.py`: Additional utility functions
- `train.py`: Training script for command-line usage
- `predict.py`: Prediction script for command-line usage
